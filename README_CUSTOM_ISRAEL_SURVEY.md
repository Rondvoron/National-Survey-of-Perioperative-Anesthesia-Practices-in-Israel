# Documentation for the Dashboard

## Overview
This document provides comprehensive details regarding the dashboard used in the National Survey of Perioperative Anesthesia Practices in Israel.

## Features
- **User-Friendly Interface**: The dashboard is designed to be intuitive and easy to navigate.
- **Data Visualization**: Provides various graphical representations of the survey data, including charts and graphs.
- **Custom Filters**: Users can filter data based on several parameters for specific insights.

## Installation
1. Clone the repository using `git clone https://github.com/Rondvoron/National-Survey-of-Perioperative-Anesthesia-Practices-in-Israel.git`
2. Navigate to the project directory.
3. Run the dashboard using the command: `npm start`

## Usage
- The dashboard allows users to upload data files in CSV format.
- Once uploaded, users can select visualization types and apply filters to view specific data points.
