# Setup Custom Israel

This is the complete setup guide for the project.

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
- You will also need [npm](https://www.npmjs.com/) for package management.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Rondvoron/National-Survey-of-Perioperative-Anesthesia-Practices-in-Israel.git
   ```
2. Navigate to the cloned directory:
   ```bash
   cd National-Survey-of-Perioperative-Anesthesia-Practices-in-Israel
   ```
3. Install the necessary packages:
   ```bash
   npm install
   ```
4. Create a .env file based on the .env.example provided:
   ```bash
   cp .env.example .env
   ```
5. Customize the environment variables in the .env file as needed.
6. Run the application:
   ```bash
   npm start
   ```
